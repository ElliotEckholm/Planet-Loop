

# Planet-Loop

Author:

1. Elliot Eckholm

Written in Lua, uses Love2d Game Engine. 

iOS app, currently under development. 

Instruction to play on iOS device:
1. Install Xcode
2. Install Love2d iOS package
3. Copy Archive.love into Xcode project
4. Run on iOS device


Archive .love file:

zip -9 -r ../Archive.love .
